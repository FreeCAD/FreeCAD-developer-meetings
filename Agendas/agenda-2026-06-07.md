6. **PSA: Recent compilation changes**
   - `-Werror` on CI jobs: pixi builds now need to be warning-free on some modules before being merged
   - pixi v7 lockfile ([pending merge](https://github.com/FreeCAD/FreeCAD/pull/30329)): will require pixi >= 0.68 to be installed. Run `pixi self-update` to automatically upgrade.
   - `gtest` and `fmt` not vendored anymore, and a few new options to provide external installation of libraries. Will provide a full 1.2 packaging changelog near the release (where to post it?)
   - In case your local build fail, try removing `.pixi` and `build` directories 
7. **Abandoned feature removal**
   - Some features are hidden behind compile flags, or are broken since the stone age.
   - Anyone wants to revive them or should they be removed for good?
   - [Stereo views](https://github.com/FreeCAD/FreeCAD/pull/30634): feature is broken
   - [Debugger](https://github.com/FreeCAD/FreeCAD/pull/30622): broken, replaced by remote debugging
   - [Oculus/Rift VR](https://github.com/FreeCAD/FreeCAD/blob/main/cMake/FindRift.cmake): uses an outdated SDK, not build in release
   - [Cloud module](https://github.com/FreeCAD/FreeCAD/pull/30563): used to connect & exchange files with AWS S3. Not enabled in release, untested.
